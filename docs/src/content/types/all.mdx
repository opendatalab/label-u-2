此处列出了一些公共类型定义。

## ISize

| Name   | Type             | Description |
| ------ | ---------------- | ----------- |
| width  | string \| number | 宽度        |
| height | string \| number | 高度        |

## IResult

| Name   | Type    | Description |
| ------ | ------- | ----------- |
| width  | number  | 宽度        |
| height | number  | 高度        |
| rotate | number  | 旋转角度    |
| step_1 | object  | TODO        |
| step_2 | object  | TODO        |
| step_3 | string  | TODO        |
| valid  | boolean | 是否有效    |

## IFile

| Name      | Type              | Description  |
| --------- | ----------------- | ------------ |
| id        | number            | 文件唯一标示 |
| filename  | string            | 文件名       |
| disabled  | boolean           | 是否失效     |
| result    | IResult \| string | 标注信息     |
| preResult | IResult \| string | 预标注信息   |
| url       | string            | 文件链接     |
| size      | number            | 文件大小     |

## IImgFile extends IFile

| Name | Type                                                     | Description |
| ---- | -------------------------------------------------------- | ----------- |
| meta | { imageWidth: number; imageHeight: number; md5: string } | 图片元信息  |
